# KeePass Diary App

> 技术栈：Electron + React (create-react-app)

## Run

```sh
# 安装依赖
npm install

# 开发模式 
npm run dev
```

## Build

```sh
# 全局安装 electron-builder
npm -i -g electron-builder

# 首先构建 React
npm run build:react

# 构建 electron 可执行文件
npm run build:electron
```
